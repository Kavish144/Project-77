Project on Console

To download Visual Studio code go to chrome and type Visual Studio Code.com. After that click on download for which ever operating system your computer or your laptop works on. Open the folder in which it is saved in and open the file to install the software. Once installed click on finish and your Visual Studio Code is ready to be used

We use console screen to check the output before showing it on the webpage. It allows you to keep track of what is going on in your code. To print anything on console your code is console.log(name of your variable);.
